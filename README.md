Smart Ride Sharing System 🚗
Project Overview

The Smart Ride Sharing System is a web-based platform that enables drivers to share available seats with passengers traveling in the same direction. The system provides ride posting, ride booking, route matching, fare calculation, payment integration, notifications, and review management.

Features
User Registration and Login
Role-Based Access (Driver & Passenger)
Secure Authentication using JWT
Ride Posting and Ride Booking
Dynamic Fare Calculation
Route Matching System
